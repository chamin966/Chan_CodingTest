투 포인터 알고리즘
  • 리스트에 순차적으로 접근해야 할 때 두 개의 점의 위치를 기록하면서 처리하는 알고리즘을
    의미한다.
  • 흔히 2, 3, 4, 5, 6, 7번 학생을 지목해야 할 때 간단히 ‘2번부터 7번까지의 학생’이라고
    부르곤 한다.
  • 리스트에 담긴 데이터에 순차적으로 접근해야 할 때는 시작점과 끝점 2개의 점으로 접근할
    데이터의 범위를 표현할 수 있다.

적용 가능 문제
  특정한 합을 가지는 부분 연속 수열 찾기
  • 𝑁 = 8개의 자연수로 구성된 수열이 있다.
  • 합이 𝑀 = 5인 부분 연속 수열의 개수를 구하여라.
  • 수행 시간 제한은 𝑂(𝑁) 이다.

• 투포인터 알고리즘의 동작 방식
  1. 시작점(𝑠𝑡𝑎𝑟𝑡)과 끝점(𝑒𝑛𝑑)이 첫 번째 원소의 인덱스(0)를 가리키도록 한다.
  2. for으로 start를 점차 증가시킨다.
  3. 내부의 while문(부분합이 목표치보다 작거나, end가 마지막 인덱스를 넘어서지 않을 경우)에서
     현재 부분 합이 𝑀보다 '작다면',
     부분합에 start 인덱스의 값을 더해주고 end를 1 증가시킨다.
  4. 현재 부분합이 M보다 같거나 크거나, end가 마지막 인덱스를 넘어서면 while문을 빠져나온다.
  5. 그때 부분합이 M과 같다면 카운트를 증가시킨다.
  6. 이제 부분합에서 start 부분을 빼고, 다시 for문의 처음으로 돌아간다.
  7. start가 end와 같아질 때까지 반복한다.
